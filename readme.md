# Reverce a list
## Изменение направления односвязного списка
Данная задача была реализована на языке java. Были использованы такие технологии:

- Jupiter
- Junit

## Структура
Сам функционал был реадизован в виде двух классов ReversedList где описываеться логика и Node в котором реализован  ноды списка. Main класс отсутствует заместь него реализованы тесты которые покрывают основные кейсы.
